## 我相信的事

> 教學不是把知識塞給學生,
> 是給他們一張地圖,讓他們知道自己在哪、要去哪。

- **陪伴比講解重要** —— 願意陪一個學生想透一題,勝過教完十題
- **結構比技巧重要** —— 知道概念怎麼串起來,比會解某一題重要
- **理解比刷題重要** —— 一題真懂,勝過十題會做
